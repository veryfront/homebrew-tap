# Homebrew Tap for Veryfront

This is the official Homebrew tap for [Veryfront](https://veryfront.com), a zero-config React meta-framework for AI-native applications.

## Installation

```bash
brew install veryfront/tap/veryfront
```

Or add the tap first:

```bash
brew tap veryfront/tap
brew install veryfront
```

## Updating

```bash
brew upgrade veryfront
```

## Other Installation Methods

- **curl (macOS/Linux):** `curl -fsSL https://veryfront.com/install.sh | sh`
- **npm:** `npm install -g veryfront`
- **npx:** `npx veryfront`
